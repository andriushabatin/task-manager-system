Как запустить проект:

    1. Скачать репозиторий;

    2. Открыть проект;

    3. Запустить контейнер с помощью docker-compose.yml
и конфигурацией из файла src/main/resources/application.properties;

    4. Поднять тестовую БД с конфигурацией из файла src/test/resources/application.properties;

    5. Запустить проект.


*Документация будет доступна по адресу http://localhost:8080/swagger-ui/index.html (ввести /v3/api-docs
 и нажать explore)
